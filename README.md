# Kawaii Deauther Pro V.2026 🚀

Perangkat lunak pengujian penetrasi WiFi profesional yang dirancang untuk melakukan gangguan (*deauthentication*) pada klien/router WiFi dan melakukan *spamming* Access Point palsu secara masif.

## ✨ Fitur Utama
* **Auto-Scan Interface:** Mendeteksi kartu WiFi secara otomatis.
* **MDK4 Engine:** Menggunakan teknologi injeksi paket terbaru yang mendukung router modern.
* **Multi-Platform:** Lancar dijalankan di Termux (Root), Kali Linux, dan Parrot OS.
* **Stealth Mode:** Mengubah MAC Address secara otomatis sebelum melakukan serangan.


## 🛠️ Persyaratan
1.  **Akses Root** (Wajib).
2.  **Monitor Mode Support:** Pastikan adapter WiFi Anda mendukung mode monitor.
3.  **Untuk Termux:** Wajib menggunakan HP yang sudah di-ROOT dan disarankan menggunakan WiFi Adapter USB eksternal via OTG.
